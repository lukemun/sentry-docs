## Modifying Default Integrations

To disable system integrations, set `defaultIntegrations: false` when calling `init()`.

To override their settings, provide a new instance with your config to the `integrations` option. For example, to turn off console capturing:

```javascript
import * as Sentry from '@sentry/nestjs';

Sentry.init({
  dsn: "___PUBLIC_DSN___",
  
  integrations: [
    Sentry.consoleIntegration({
      levels: ['warn', 'error'] // Only capture warnings and errors
    }),
  ],
});
```
